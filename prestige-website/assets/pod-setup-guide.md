# Prestige Group — Print-on-Demand Setup Guide

## Step 1: Set Up Your Accounts (You do this, 30mins)

### Etsy Shop
1. Go to etsy.com/sell and create a shop
2. Shop name ideas: LondonDrift, BoroughThreads, ThamesThreads, LDNPrints
3. Set currency to GBP
4. Choose "Apparel & Accessories" as category
5. Upload shop banner (I'll create this)
6. Write shop bio

### Printify Account (FREE)
1. Go to printify.com and sign up (free)
2. Connect to your Etsy shop via Printify dashboard
3. This is the key — Printify handles ALL printing + shipping automatically

### Canva Pro (£10.99/month)
1. Upgrade to Canva Pro for premium templates + transparent PNG exports
2. Use my designs or create your own
